## 前言

本项目为个人博客项目，目标是实现一个前后端分离的个人博客

## 启动命令

安装依赖：pnpm install<br>

前端：pnpm run dev:blogs<br>

后端：pnpm run dev:service<br>

## 技术栈

包管理：pnpm monorepo <br>

前端： vue3、pinia、ts、vite<br>

后端： koa2、sequalize-typescript、ts、mysql5.7<br>
